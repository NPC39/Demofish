GitHub Pages Demo Notes
=======================

This version is static-only and safe for GitHub Pages.

What it does:
- Accepts any text in the username and password fields.
- Saves a demo record in browser localStorage.
- Displays only the placeholder: * raw data redacted *
- Never displays the original typed value back in the view page.

How to publish:
1. Upload all files in this folder to a GitHub repository.
2. Go to Settings > Pages.
3. Choose Deploy from a branch.
4. Select main and /(root).
5. Save and wait for the site URL.

Pages included:
- index.html
- success.html
- view_demo.html
- assets/style.css
- assets/script.js
